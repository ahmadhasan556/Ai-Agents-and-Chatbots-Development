# 🤖 AI Agents – SkillUstad Learning Repository

This repository contains structured learning material and practical implementations for **AI Agents, Python, and modern AI development tools**. It is designed for **SkillUstad students** to learn step-by-step how to build intelligent applications using Python, LLM frameworks, APIs, and agent systems.

The goal of this repository is to move from **basic Python → AI tools → real AI agents**.

---


# 🚀 How to Use This Repository

1. Clone the repository

```
git clone <repo-url>
```

2. Navigate into a specific module

```
cd 01_PYTHON
```

3. Install dependencies

```
pip install -r requirements.txt
```

4. Run the project or scripts.
